# 画布格式规范

> SVG 基础规则见 shared-standards.md。

## 格式速查表

| 格式         | viewBox         | 比率      | 用途                 |
| ------------ | --------------- | --------- | -------------------- |
| PPT 16:9     | `0 0 1280 720`  | 16:9      | 商务演示、会议       |
| PPT 4:3      | `0 0 1024 768`  | 4:3       | 传统投影仪、学术演讲 |
| 小红书       | `0 0 1242 1660` | 3:4       | 图文分享、知识帖     |
| 朋友圈 / IG  | `0 0 1080 1080` | 1:1       | 方形海报、品牌展示   |
| Story / 抖音 | `0 0 1080 1920` | 9:16      | 竖版故事、短视频封面 |
| 公众号封面   | `0 0 900 383`   | 2.35:1    | 微信公众号文章头图   |
| 横版 Banner  | `0 0 1920 1080` | 16:9      | 网页横幅、数字屏幕   |
| 竖版海报     | `0 0 1080 1920` | 9:16      | 手机屏幕、电梯广告   |
| A4 印刷      | `0 0 1240 1754` | 1:sqrt(2) | 印刷海报、传单       |

## 格式选择决策树

```
内容用途？
├── 演示文稿
│   ├── 现代设备 → PPT 16:9 (1280x720)
│   └── 传统设备 → PPT 4:3 (1024x768)
├── 社交分享
│   ├── 小红书 → 1242x1660
│   ├── 朋友圈 / IG → 1080x1080
│   └── Story / 抖音 → 1080x1920
└── 营销物料
    ├── 公众号封面 → 900x383
    ├── Banner → 1920x1080
    └── 印刷 → 1240x1754
```

## 布局原则

### 横版（16:9、4:3、2.35:1）
- 视觉流向：Z 字型，从左到右
- 边距：40-80px
- 布局：多栏、左右分割、网格
- 卡片尺寸（16:9）：单行 530-600px，双行 265-295px

### 竖版（3:4、9:16）
- 视觉流向：从上到下
- 边距：60-120px
- 布局：单栏、上下分割、卡片堆叠
- 卡片尺寸（3:4）：高度 400-600px，间距 40-60px

### 方形（1:1）
- 视觉流向：中心放射
- 边距：60-100px
- 核心区域：约 800x800px

## 格式特定设计

| 格式       | 标题区域            | 内容区域       | 特殊说明                     |
| ---------- | ------------------- | -------------- | ---------------------------- |
| PPT        | 80-100px            | 全宽利用       | 页码右下角                   |
| 小红书     | 180-240px（加粗）   | 上下留白充足   | 底部品牌区 120-160px         |
| 朋友圈     | 200-280px           | 中部 500-600px | 底部二维码区 150-200px       |
| Story      | —                   | 中部 1500px    | 顶部安全区 120px，底部 180px |
| 公众号封面 | 居中/左对齐 48-72px | —              | 图片在右侧或作为背景         |

## ViewBox 示例

```xml
<svg width="1280" height="720" viewBox="0 0 1280 720">   <!-- PPT 16:9 -->
<svg width="1242" height="1660" viewBox="0 0 1242 1660"> <!-- 小红书 -->
<svg width="1080" height="1080" viewBox="0 0 1080 1080"> <!-- 朋友圈 -->
<svg width="1080" height="1920" viewBox="0 0 1080 1920"> <!-- Story -->
<svg width="900" height="383" viewBox="0 0 900 383">     <!-- 公众号封面 -->
```
